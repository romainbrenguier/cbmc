Suggestion Box
~~~~~~~~~~~~~~
Suggestion Box is an Web application for submitting the suggestions by Users in a company.
7 commits
1 branch
0 releases
1 contributor


Repository: https://github.com/Tirrumala/SuggestionBox

Installation on Ubuntu:

1. Open a terminal in the directory of this readme file and clone:
      git clone https://github.com/Tirrumala/SuggestionBox
2. Enter the directory and build with maven:
      cd SuggestionBox/SuggestionBox/
      git checkout  a55074e07008c1a50042f472694d465db182d874 .
      mvn clean package
3. Copy the resulting files to the installation directory:
      cp -r "./target/" "../../../BENCHMARK"
